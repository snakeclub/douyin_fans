连接手机帮助
通过USB连接手机步骤
1. 打开手机的 设置 -> 系统 -> 开发人员选项 -> 调试，设置为 USB 调试模式;
2. 通过 USB 数据线连接手机;
3. 通过 “自动添加设备” 或 “手工添加设备” 按钮连接手机。

 

通过WIFI连接手机步骤
1. 打开手机的 设置 -> 系统 -> 开发人员选项 -> 调试，设置为 USB 调试模式;
2. 通过 USB 数据线连接手机;
